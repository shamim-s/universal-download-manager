# UDM Desktop UI (Tauri + React)

This folder will hold the Tauri desktop app. In **Phase 0**, generate the project scaffold here:

```bash
cd ui
npm create tauri-app@latest .      # choose React + TypeScript
npm install
npm install zustand @tanstack/react-query recharts lucide-react
npm run tauri dev
```

The stub files under `src/` describe the intended structure (Phase 8). When you run
`create-tauri-app`, merge these into the generated project (or use them as a reference).

## Structure (target)

```
ui/
├── src/
│   ├── App.tsx
│   ├── types.ts                 # mirrors daemon protocol
│   ├── hooks/useDaemonSocket.ts # WS to ws://127.0.0.1:60123
│   ├── store/downloads.ts       # Zustand store
│   └── components/
│       ├── DownloadList.tsx
│       ├── DownloadItem.tsx
│       ├── Toolbar.tsx
│       ├── SpeedGraph.tsx
│       └── Settings.tsx
├── src-tauri/                   # generated by create-tauri-app (tray, config)
└── package.json                 # generated by create-tauri-app
```
